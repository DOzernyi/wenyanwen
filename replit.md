# 閱讀古文 (Classical Chinese Reading)

## Overview
A Jekyll-based static website for reading and studying classical Chinese texts. Uses the Jekyll Minimal theme.

## Project Structure
- `_layouts/` - HTML layout templates
- `_includes/` - Reusable HTML snippets
- `_sass/` - SCSS stylesheets
- `assets/` - Static assets (CSS, JS, images)
- `pages/` - Content pages in multiple languages
- `白話翻譯/` - Vernacular Chinese translations

## Technology Stack
- Ruby 3.2
- Jekyll 4.x
- Jekyll Minimal Theme

## Development
Run locally with:
```
bundle exec jekyll serve --host 0.0.0.0 --port 5000 --livereload
```

## Deployment
Static site deployment with Jekyll build. Output goes to `_site/` directory.
